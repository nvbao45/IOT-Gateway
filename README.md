<!-- GETTING STARTED -->
## Getting Started


### Prerequisites

* python 3.6+
* pip

### Installation

1. Clone the repo
   ```sh
   https://github.com/nvbao45/IOT-Gateway.git
   ```
2. Install packages
   ```sh
   pip install -r requirements-mysql.txt
   ```
3. Run flask app
   ```js
   flask run
   ```
